# Arivolam / Campusolam: Project Overview

## 1. Introduction
Arivolam (featuring the Campusolam ERP) is a comprehensive, next-generation educational resource planning and campus management system. It is designed to streamline administrative operations, enhance faculty-student interaction, and provide a unified digital platform for academic management.

## 2. Core Modules
- **Dev-Admin Dashboard:** Superuser control for core platform settings, environment configurations, and global management.
- **Admin Dashboard:** Centralized control for managing users, academic calendars, and system resources.
- **Faculty Portal:** Tools for attendance tracking, marks entry, and academic performance monitoring.
- **Student Portal:** Real-time access to attendance, assignments, grades, and timetables.
- **Interactive Campus Map:** An interactive, GIS-based campus exploration tool allowing users to navigate campus facilities seamlessly.

## 3. Technology Stack
- **Frontend Framework:** Next.js (React 19)
- **Styling & UI:** Tailwind CSS, Radix UI Primitives
- **Animations:** Framer Motion, GSAP
- **Mapping & GIS:** Leaflet, React-Leaflet, Turf.js
- **Backend & Database:** Supabase (PostgreSQL, Authentication, Row Level Security)
- **Deployment:** Vercel (or Custom CI/CD Auto-Deploy)

## 4. Key Innovations
- **Server-Side Rendering (SSR):** Leveraging Next.js App Router for optimal performance.
- **Real-time Database:** Supabase ensures instant updates to attendance and marks.
- **Role-Based Access Control (RBAC):** Secure data boundaries between students, faculty, administrators, and dev-admins.
- **Modern UI/UX:** Clean, intuitive interface with comprehensive accessibility standards.

## 5. Future Enhancements
- **3D Campus Navigation:** Interactive 3D maps with building walk-throughs and wayfinding.
- **AR/VR Immersive Tours:** Virtual reality campus tours and augmented reality enhancements.
